Note: If you make any alterations to the report result columns or the report file type (i.e. CSV, FixedWidth, etc.), this file format will no longer be valid. Be sure to save this report file type before you exit the General tab.

*"<Today's Date>" will be the only variable in the file name that will change and ensure your file names will be unique each time the report is ran. This date will be the date the report is ran and will be in the format "MMddyyHHmmss".
